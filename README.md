# Project Carb — Dashboard

A project management dashboard for Project Carb. Built so teammembers have the least amount of friction possible to:

1. Do their assigned tasks
2. Find work on their own — easier and even motivating
3. Find and maintain devotion to the vision "The Grammarly of Additive Manufacturing" 

## Current state

This dashboard will be the source of truth, not a view on top of an existing system. There exists a wealth of code, documents, and other loose files associated with the project. 

## Task areas (non-exhaustive, just currently known)

- Model training pipeline (physically printing, scanning, processing objects)
- Backend code development (modules, model improvement, training)
- Frontend (branding, UI, GUI, website, this dashboard itself)
- Marketing and sales

These are not necessarily MECE.

## Other docs

- `DESIGN.md` — design decisions, with reasoning
- `REJECTED.md` — paths considered and rejected, with reasoning. Read this before proposing alternatives so the same ideas aren't re-litigated.
- `OPEN_QUESTIONS.md` — what's not decided yet
- `SKETCHES.md` — descriptions of the layouts that were sketched during design
